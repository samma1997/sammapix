## PART 1: UNDERSTANDING BACKLINK ANALYSIS

### What is a Backlink?
A backlink (inbound link) is when another website links to your site.

**Example:**
- Blogging Wizard writes: "The best image compression tools are: TinyPNG, SammaPix, and Squoosh"
- This creates a backlink FROM bloggingwizard.com TO sammapix.com

**Why it matters:**
1. **SEO Signal:** Google sees backlinks as "votes of confidence"
2. **Authority:** Links from high-authority sites (DA 60+) = more SEO power
3. **Traffic:** People click links and visit your site
4. **Ranking:** More quality backlinks = higher rankings for related keywords

### Types of Backlinks

**Dofollow links** (Good)
- Pass SEO authority
- Google counts them
- Example: `<a href="sammapix.com">image tool</a>`

**Nofollow links** (Less good but still valuable)
- Don't pass SEO authority
- But still drive traffic and build brand awareness
- Example: `<a href="sammapix.com" rel="nofollow">image tool</a>`

**For your strategy:** Target 70% dofollow, 30% nofollow okay.

## PART 2: HOW TO ANALYZE COMPETITORS (FREE TOOLS)

### Tool #1: Google Search Console (GSC) — 100% Free
**Best for:** Monitoring your own backlinks (once you start getting them)

**How to use:**
1. Log in to Google Search Console (search.google.com/search-console)
2. Select your property (sammapix.com)
3. Left sidebar → Links
4. You'll see:
   - "Top Linking Sites"
   - "Top Linked Pages"
   - New links you've gained

**Action:** Set up GSC right now if not done yet. Come back here after Month 1 to monitor growth.

---

### Tool #2: Ahrefs Free Backlink Checker
**Best for:** Analyzing competitor backlinks (quick, reliable data)

**URL:** ahrefs.com/backlink-checker

**How to use:**

1. **Go to Ahrefs Backlink Checker**
2. **Enter domain:** tinypng.com
3. **Wait for results** (takes 10–30 seconds)
4. **You see:**
   - Total backlinks
   - Referring domains
   - Top backlinks by domain authority
   - Links by anchor text

**What to look for:**

```
Imagine you see:
- bloggingwizard.com → TinyPNG (backlink with DA 67)
- theadminbar.com → TinyPNG (backlink with DA 54)
- enviragallery.com → TinyPNG (backlink with DA 52)
- photoworkout.com → TinyPNG (NO backlink)

Action:
"photoworkout.com wrote about image tools and linked TinyPNG
but NOT SammaPix. This is an opportunity."
```

**How to export data:**
1. Click "**Export as CSV**" button
2. Save to Google Drive
3. Open in Google Sheets
4. Filter by Domain Authority (keep only 40+)
5. Manually visit each site to verify it's a relevant "tools" article

### Tool #3: Ubersuggest Free Backlink Tool

"free image optimization tools" site:*.com

## PART 4: THE "LINK OPPORTUNITY SCORE" FORMULA

### Score each site 1–10 (higher = more valuable)

**Use this formula:**

```
Opportunity Score = (Domain Authority / 10) + Topic Relevance + Contact Quality

Domain Authority (0–10 points):
- DA 60+: 8–10 points
- DA 40–59: 5–7 points
- DA 20–39: 2–4 points
- DA < 20: 0–1 points

Topic Relevance (0–10 points):
- Direct image tool fit: 10 points
- Image tools + photography: 8 points
- Web performance / SEO: 6 points
- Content creation general: 4 points
- Unrelated: 0 points

Contact Quality (0–10 points):
- Direct email address: 8 points
- Social media (Twitter, LinkedIn): 6 points
- Contact form: 5 points
- No contact info: 0 points
```

**Example:**
```
PhotoWorkout:
- DA 38: 4 points
- Topic relevance (image tools + photography): 8 points
- Contact quality (Twitter + email): 7 points
= SCORE: 19/30 (HIGH priority)

Blogging Wizard:
- DA 67: 9 points
- Topic relevance (image tools): 10 points
- Contact quality (no email, refuses pitches): 0 points
= SCORE: 19/30 (HIGH potential but LOW feasibility)
```

## PART 5: FINDING CONTACT INFORMATION

### Where to Find Emails

**Tier 1 (Easiest):** Author page on the website
```
Example: bloggingwizard.com/author/adam-connell
Usually shows: email, Twitter, LinkedIn
```

**Tier 2:** Contact page
```
Go to: domain.com/contact
Usually has contact form or email
```

**Tier 3:** Social media
```
Twitter/X: Find author's profile
LinkedIn: Search author name
Bluesky: New platform, sometimes has direct messages
```

**Tier 4:** Hunter.io (Paid, but $49/month gives unlimited lookups)
```
Website: hunter.io
Enter: domain.com
Shows: emails associated with that domain
```

**Tier 5:** Google (Free, manual)
```
Search: "adam connell email"
Or: "adam connell contact"
```

### Pro Tip: LinkedIn
```
1. Search author name on LinkedIn
2. Click their profile
3. Often shows email in "About" section or DM message
4. Even if no email visible, can DM them directly
```

## PART 7: RED FLAGS — SITES TO SKIP

### Don't waste time on these:

**❌ Explicitly refuses pitches**
```
Example: Blogging Wizard contact page states:
"We don't accept content update or link requests"
Action: Skip entirely
```

**❌ Spammy/low quality**
```
Signs:
- Lots of ads everywhere
- Broken links on page
- Poor writing quality
- DA < 20
Action: Skip (link won't help much)
```

**❌ Unrelated niche**
```
Example: "Best Image Tools for Medical AI"
Problem: Wrong audience, unnatural link
Action: Skip
```

**❌ Affiliate/paid placement only**
```
Signs:
- "Link pricing: $300 per placement"
- All tools have "/ref/" or affiliate links
- No editorial content (pure advertorial)
Action: Only pursue if you have budget
```

## PART 11: COMPETITIVE INTELLIGENCE — What You Learn

### When you analyze TinyPNG's backlinks, you'll notice:

**Pattern 1: Listicle Sites**
- 60% of links come from "best tools" articles
- These sites accept 5–15 tools per article
- High turnover = updatable content
- **Opportunity:** Request to be added when they update

**Pattern 2: WordPress Blogs**
- 20% of links are from WordPress-focused blogs
- They care about page speed + SEO
- Often mention TinyPNG as plugin or tool
- **Opportunity:** Create content about WordPress + image optimization

**Pattern 3: Tech News**
- 10% from TechCrunch, ProductHunt, HN
- Only major announcements get covered
- Hard to get, but high-value
- **Opportunity:** Launch with interesting angle (AI rename, batch processing)

**Pattern 4: Directory Sites**
- 10% from AlternativeTO, G2, Capterra, etc.
- Easy to get (free submission)
- Low individual value, high aggregate value
- **Opportunity:** Submit to ALL directories immediately

### When you notice the gaps:

```
"TinyPNG gets linked by listicle sites, but:
- No sites link to it specifically for 'batch processing'
- No sites mention 'privacy-first'
- No sites focus on 'HEIC conversion'

So SammaPix can target these underserved angles."
```
