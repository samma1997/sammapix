## ❓ FAQ

### Q: Can Playwright really record video automatically?
**A:** Yes. Playwright has native `recordVideo` context option. Saves as WebM. No FFmpeg needed for WebM format.

### Q: Do I need FFmpeg?
**A:** Only if you want MP4 output. WebM works fine for YouTube. `brew install ffmpeg` if needed.

### Q: What's the quality like?
**A:** 1280x720 (HD) minimum. Professional demo quality. Comparable to Arcade or Loom recordings.

### Q: Can I add voiceover automatically?
**A:** Yes. Use Synthesia API or merge with FFmpeg. Scripts provided for both.

### Q: How do I know if my selectors are correct?
**A:** Test in browser console: `document.querySelector('your-selector')` should return the element.

### Q: What if Playwright selector fails?
**A:** Use Arcade.software instead (doesn't require selectors). Or check `CSS_SELECTOR_REFERENCE.md` for debugging.

### Q: Can I schedule YouTube uploads?
**A:** Yes. Use `scheduledStartTime` in `youtube-bulk-uploader.js` to schedule publication date/time.

### Q: Will these videos rank on YouTube?
**A:** Yes. With proper SEO metadata (provided in templates), expect ranking within 2-4 weeks.

## 🚨 Common Issues & Solutions

### Issue: "Playwright not found"
**Solution:** `npm install playwright` (requires Node.js)

### Issue: "CSS selector not working"
**Solution:** Check `CSS_SELECTOR_REFERENCE.md` → Debugging section

### Issue: "Video file too large"
**Solution:** `ffmpeg -i input.mp4 -crf 28 output.mp4` (higher CRF = smaller)

### Issue: "YouTube says 'processing'"
**Solution:** Wait 24 hours for large files. Check in YouTube Studio.

### Issue: "Selectors for my tool differ"
**Solution:** Open tool → F12 → inspect elements → update script
