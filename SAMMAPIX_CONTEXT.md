# SammaPix — Complete Context for Marketing Execution

## What is SammaPix
SammaPix (sammapix.com) is a browser-based AI photo and file tool platform. 20 tools for image optimization, AI-powered organization, and file management. All processing happens client-side (files never leave the user's browser). Privacy-first.

## Owner
Luca Sammarco — Italian photographer and web developer. Personal brand: lucasammarco.com

## Tools (20 total)
### Optimize (6)
- **Compress** — Shrink JPG, PNG, WebP, GIF up to 90%
- **WebP Converter** — Convert any image to WebP
- **HEIC Converter** — iPhone HEIC to JPG/WebP
- **PDF to Image** — Convert PDF pages to JPG/PNG/WebP
- **Batch Resize** — Resize for Instagram, Twitter, LinkedIn presets
- **Crop & Ratio** — Crop to 1:1, 16:9, 4:3, A4, etc.

### AI-Powered (6)
- **AI Rename** — AI generates SEO-optimized filenames (Gemini Flash)
- **AI Alt Text** — Generate accessibility alt text with AI
- **AI Organize** (NEW, killer feature) — Drop ANY files (images, PDFs, Word, Excel, video, audio, code). AI sorts into smart folders, finds duplicates, renames based on content. NO competitor does this browser-based.
- **AI Photo Sort** — AI categorizes images automatically
- **Transcribe** — AI transcription with timestamps, SRT subtitles
- **Web Optimize** — Compress + WebP + AI rename in one click

### Creative (2)
- **Film Filters** — 14 analog film presets (Kodak Gold, Fuji, Ilford)
- **Watermark** — Batch watermark with text or logo

### Organize (6)
- **EXIF Viewer** — Strip GPS, camera data, metadata
- **Find Duplicates** — Perceptual hashing finds exact/near dupes
- **Sort by Location** — Sort photos by country using GPS EXIF
- **Photo Map** — Interactive map from travel photos
- **Cull** — Rate and cull a shoot with star ratings
- **Batch Rename** — Rename files with custom pattern
