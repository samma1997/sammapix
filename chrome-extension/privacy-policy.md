# Privacy Policy — SammaPix Chrome Extension

**Last updated:** April 12, 2026

## What this extension does

SammaPix Chrome Extension provides quick access to browser-based image tools (compress, resize, convert, remove background, strip EXIF data, passport photos).

## Data collection

**This extension collects NO data.** Specifically:

- No images are uploaded to any server
- No browsing history is collected
- No personal information is stored
- No cookies are set
- No analytics or tracking of any kind
- No data is shared with third parties

## How it works

When you click a tool in the extension popup or right-click an image, the extension opens the corresponding tool page on sammapix.com. All image processing happens entirely in your browser via Canvas API and WebAssembly. Your images never leave your device.

## Permissions used

- **contextMenus**: To add "SammaPix" to the right-click menu on images
- **activeTab**: To detect when you right-click on an image
