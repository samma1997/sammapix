## REDDIT COMMENT STRATEGY

### When to Jump Into Existing Discussions

If you see a Reddit discussion about image optimization, compression, tools, etc:

1. **Don't lead with your tool** — answer the actual question first
2. **If relevant:** "I actually built something that does this exactly — [tool]"
3. **Ask for feedback:** "Would love your thoughts" (not "please promote")
4. **Engage genuinely:** Respond to follow-up questions, address concerns
5. **Know when to back off:** Don't oversell or become defensive

### Example Response

```
Great question! The way I think about it is:

[Direct answer to their question with detail]

[If relevant:] I actually spent time on this exact problem and built a tool
that handles it — [tool link]. Curious what you think, totally happy for feedback.

One thing I'd add: [One additional insight]

Does that help?
```

## QUICK-REFERENCE POSTING CHECKLIST

### Before Publishing Blog Post
- [ ] SEO title (50-60 chars, includes keyword)
- [ ] Meta description (150-160 chars)
- [ ] Hero image (1200x600px, optimized)
- [ ] Internal links (3-5 to other blog posts)
- [ ] External links (2-3 to authority sources)
- [ ] H1, H2, H3 structure correct
- [ ] CTA button/link to SammaPix
- [ ] No spelling/grammar errors (Grammarly check)
- [ ] Formatted for scanning (short paragraphs, bullets, bold)
- [ ] Responsive on mobile
- [ ] Social preview looks good

### Before Publishing Reddit Post
- [ ] Title is compelling (no all-caps except acronyms)
- [ ] Spelling/grammar perfect (Reddit is unforgiving)
- [ ] No direct sales language
- [ ] Authentic voice (sounds like you, not marketing)
- [ ] Timing correct (9 AM EST for high-traffic subs)
- [ ] Subreddit rules followed
- [ ] Ready to monitor for first 6 hours
- [ ] Link UTM-tagged correctly
- [ ] Backup subreddit identified

### Before Sending Outreach Email
- [ ] Subject line mentions their work specifically
- [ ] Greeting spelled correctly
- [ ] Body references specific article/post they wrote
- [ ] No generic template language
- [ ] CTA is soft (not "please promote")
- [ ] Sign-off is personalized
- [ ] Spell check done
- [ ] Link works and UTM-tagged
- [ ] Send time appropriate (9-11 AM their timezone if you know it)
